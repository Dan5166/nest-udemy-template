# Teslo API
1. Instalar nest config
2. Configurar el app.module.ts con el nest config
3. Levantar la Base de datos
```
docker-compose up -d
```
4. Instalar los validadores de clases
5. Pegar los global Pipes
6. npm i uuid
7. npm i -D @types/uuid
8. Ejecutar seed
```
http://localhost:3000/api/seed
```
9. Levantar npm run start:dev
